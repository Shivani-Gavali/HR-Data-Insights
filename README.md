# HR Metrics
# Power BI Project

HR Data Insights is an interactive Power BI dashboard built to analyze workforce trends, employee demographics, and attrition patterns.
It helps HR teams make data-driven decisions to improve retention, employee satisfaction, and organizational performance.

# Tools Covered : Advanced Excel, Power BI, DAX, Power Query

# Key Performance Indicators (KPIs):
• Attrition Rate

• Attrition Count

• Total Employees

• Active Employees

• Average Employees

• Total Male

• Total Female

• Department wise Attrition Rate

• Employees by Age Group

• Gender wise Attrition Rate

• Age Group wise Attrition Rate

• Education Field wise Attrition Count

• Education Field wise Total Employees

• Department wise Total Employees

• Department wise Active Inactive Employees
